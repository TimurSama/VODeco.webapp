# Экономические расчеты VOD

## Структура папки

### 1. Базовые расчеты
- `water_economic_model.txt` - Экономическая модель водных ресурсов
- `tokenomics.txt` - Токеномика VOD
- `tokenomicFULL.txt` - Полная токеномика с расчетами

### 2. Региональная экономика
- `regional_economy/` - Расчеты по регионам
  * `asia/` - Азия
  * `europe/` - Европа
  * `america/` - Америка
  * `africa/` - Африка
  * `oceania/` - Океания

### 3. Секторальная экономика
- `sectors/` - Расчеты по секторам
  * `agriculture/` - Сельское хозяйство
  * `industry/` - Промышленность
  * `household/` - Бытовой сектор
  * `infrastructure/` - Инфраструктура

### 4. Финансовые инструменты
- `financial/` - Финансовые расчеты
  * `investments/` - Инвестиции
  * `roi/` - ROI проектов
  * `trading/` - Торговля
  * `derivatives/` - Производные инструменты

### 5. Социальная экономика
- `social/` - Социальные расчеты
  * `health/` - Здравоохранение
  * `education/` - Образование
  * `employment/` - Занятость
  * `living_standards/` - Уровень жизни

### 6. Экологическая экономика
- `ecology/` - Экологические расчеты
  * `water_quality/` - Качество воды
  * `climate/` - Климат
  * `biodiversity/` - Биоразнообразие
  * `sustainability/` - Устойчивое развитие

## Правила работы с файлами

1. Все расчеты должны быть документированы
2. Использовать единые единицы измерения
3. Указывать источники данных
4. Обновлять расчеты при изменении исходных данных
5. Проверять взаимосвязи между разными разделами

## Форматы файлов

- `.txt` - Текстовые описания и расчеты
- `.md` - Документация в формате Markdown
- `.csv` - Табличные данные
- `.json` - Структурированные данные
- `.xlsx` - Сложные расчеты и графики
